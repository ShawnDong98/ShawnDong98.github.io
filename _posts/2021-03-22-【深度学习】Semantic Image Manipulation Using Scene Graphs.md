# Abstract

图像处理可以被认为是图像生成的特殊情况，其中要生成的图像是对现有图像的修改。在大多数情况下，图像生成和处理都是对原始像素进行操作的任务。但是，在学习丰富的图像和对象表示形式方面取得的显着进步为诸如文本到图像或布局到图像生成等主要由语义驱动的任务打开了道路。在我们的工作中，我们解决了来自场景图的图像处理的新问题，在该问题中，用户可以仅通过应用从图像生成的语义图的节点或边缘中的更改来编辑图像。我们的目标是在给定的星型图中对图像信息进行编码，然后从中生成新的星型图，例如替换对象，甚至更改对象之间的关系，同时尊重原始图像的语义和样式。我们引入了一个空间语义图网络，该网络不需要直接监督星型图变化或图像编辑。这使得从现有的现实世界数据集中训练系统成为可能，而无需额外的注释工作。
