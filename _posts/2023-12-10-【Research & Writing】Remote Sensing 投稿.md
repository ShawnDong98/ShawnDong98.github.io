# Abstract

不超过200字。对于研究文章，摘要应该提供工作的相关概述。我们强烈建议作者使用以下结构化摘要的风格，但不需要标题：（1）背景：将所讨论的问题置于广泛背景之中，并强调研究的目的；（2）方法：简要描述主要方法或者处理方式；（3）结果：总结文章的主要发现；（4）结论：指出主要的结论或解释。摘要应该是对文章的客观描述，它不应该包含没有在正文中提出和证实的结果，也不应该夸大主要结论。

# 0. How to Use this Template

模板详细说明了可以在稿件中使用的各个部分。请注意，文章各部分的顺序和名称可能与期刊的要求不同（例如，材料和方法部分的位置）。请检查期刊作者页面上的说明，以核实正确的顺序和名称。如有任何疑问，请联系期刊的编辑办公室或support@mdpi.com。关于LaTeX相关问题，请联系latex@mdpi.com。


# 1. Introduction

研究引言部分应简要地将研究置于一个广阔的背景下，并突出其重要性。它应界定工作的目的及其意义。应仔细回顾当前研究领域的状态，并引用关键文献。必要时，请突出有争议和不同的假设。最后，简要提及工作的主要目标并强调主要结论。请尽可能使引言部分对您特定研究领域之外的科学家也易于理解。引用期刊论文时，请用[1]。现在引用书籍参考文献[2,3]或其他类型的参考文献[4-6]。请使用指令[7,8]引用以下使用作者-日期引用方式的MDPI期刊：行政科学、艺术、计量经济学、经济学、家谱、人文学科、 IJFS、智力研究杂志、新闻与媒体、JRFM、语言、法律、宗教、风险、社会科学、文学。


# 2. Materials and Methods

材料与方法部分应详细描述，以使其他人能够复制并在已发表结果的基础上进一步研究。请注意，您的稿件一旦发表，即表示您必须向读者提供与出版物相关的所有材料、数据、计算机代码和协议。若材料或信息的可获得性有任何限制，请在提交阶段披露。新方法和协议应详细描述，而成熟的方法可以简要描述并给出适当的引用。

报道大型数据集的研究手稿，若数据已存放在公开可用的数据库中，应指明数据的存放位置，并提供相关的访问号码。如果在提交时尚未获得访问号码，请声明将在审查期间提供。必须在发表前提供。

涉及动物或人类的干预性研究，以及其他需要伦理批准的研究，必须列出授予批准的机构和对应的伦理批准代码。这是一句引用的例子。

# 4. Discussion

作者应该讨论结果以及如何从先前研究的视角和工作假设中解释这些结果。研究发现及其含义应尽可能在广泛的背景下讨论。未来的研究方向也可以被强调。

# 5. Conclusions

这一部分并非必须的，但是如果讨论部分异常冗长或复杂，可以增加到稿件中。

# 6. Patents

该部分并非强制性内容，但如果本手稿报告的工作产生了专利，可以添加此部分。

**作者贡献：** 对于有多位作者的研究文章，必须提供一个简短的段落来说明他们各自的贡献。应该使用以下陈述：“概念化，X.X.和Y.Y.；方法论，X.X.；软件，X.X.；验证，X.X.、Y.Y.和Z.Z.；形式化分析，X.X.；调查研究，X.X.；资源，X.X.；数据整理，X.X.；写作-初稿准备，X.X.；写作-审查与编辑，X.X.；可视化，X.X.；监督，X.X.；项目管理，X.X.；资金获取，Y.Y.。所有作者已阅读并同意发表的稿件版本。”请参照CRediT分类法以解释术语。作者身份必须限于那些对所报道工作有实质性贡献的人。 资助：

**基金：** 请添加：“本研究没有接受外部资助”或者“本研究由资助者名称提供的资助，授予编号XXX。”以及“文章处理费用由XXX资助”。仔细检查提供的详细信息是否准确，并在https://search.crossref.org/funding 上使用资助机构名称的标准拼写，任何错误都可能影响到您未来的资金支持。


**机构审查委员会声明:** 在本节中，您应当添加机构审查委员会声明及批准编号（如果和您的研究相关）。如果您的研究不需要伦理审批，您可以选择不包括此声明。请注意，编辑部可能会要求您提供更多信息。对于涉及人类的研究，请添加“本研究已依照《赫尔辛基宣言》进行，且已由XX机构的机构审查委员会（或伦理委员会）批准（协议代码XXX，及批准日期）。”或者对于动物研究，请添加“动物研究协议已由XX机构的机构审查委员会（或伦理委员会）批准。”

**利益冲突声明：** 声明利益冲突或者声明“作者无任何利益冲突。”作者必须识别并声明任何可能被视为不当影响报告研究结果的表述或解释的个人情况或利益。资金提供方在研究设计；在数据收集、分析或解释；在手稿撰写；或在决定发布结果上的任何作用都必须在此部分声明。如果没有任何作用，请声明“资金提供方在研究设计；在数据收集、分析、或解释；在手稿撰写；或在决定发布结果上无任何作用”。
