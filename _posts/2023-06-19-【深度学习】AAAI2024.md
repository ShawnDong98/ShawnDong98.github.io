# Main Technical Track: Call for Papers

## Summary

- 三个技术 tracks (Main Track; 人工智能促进社会影响;安全、稳健和负责任的人工智能)
- 两阶段评审:第一阶段进行两次评审，第二阶段对第一阶段未被拒绝的论文进行额外评审。第二阶段后的作者回复，仅适用于第一阶段未被拒绝的论文。
- 所有论文可以同时提交三种补充材料:(1)技术附录;(2)多媒体;(3)代码和数据。补充材料截止日期为论文提交截止日期后3天。
- 附加页面仅供参考。
- 所有作者必须完成可重复性检查表，这有利于报告研究的重现。
- 除非有特殊情况，所有的作者都应该可以参加评审(轻负荷)。
- NeurIPS-23快速通道:最终平均分数至少为4.9分的被拒论文可以直接提交到AAAI-24的第二阶段，同时提交之前的审稿和论文ID。

AAAI系列会议的目的是促进人工智能(AI)的研究，并促进整个人工智能及其附属学科的研究人员、从业者、科学家、学生和工程师之间的科学交流。AAAI-24是第三十八届AAAI人工智能会议。与AAAI-23一样，本次会议的主题是在人工智能内部和外部建立合作桥梁。除了桥梁主题，我们还强调人工智能对社会影响和负责任的人工智能的重要性。与AAAI 2023会议一样，AAAI-24将包括技术论文演讲、特别曲目、特邀 track、研讨会、教程、海报会议、高级成员演讲、比赛和展览项目，以及另外两项活动: Bridge Program 和  Lab Program。许多活动都是根据 Bridge 的主题量身定制的，并根据最高标准进行选择，为学生和年轻研究人员提供额外的项目。

我们计划AAAI-24是一个面对面的会议。但是，鉴于局势多变，会议的确切形式只能在稍后阶段决定。


## Collaborative Bridge Theme

在其学科多样性的推动下，AAAI孵化了许多人工智能子学科和会议，几十年来培养了人工智能的凝聚力。为了探索新的机会和观点，当两个或更多的学科走到一起时，新的社区就会出现;如今，两者都很丰富。今年的 “Bridge Program” 的目的是通过培养两个或多个社区之间的合作，为共同的目标挖掘新的创新来源。我们对 Bridge 的理解是广泛的，涵盖了人工智能内外的学科。因此，我们的 Bridge Program 旨在汇集的社区可以是人工智能的不同子领域，例如规划和学习，或者是为人工智能做出贡献并从中受益的不同学科，例如人工智能和人文学科。


## Topics

AAAI-24欢迎提交关于人工智能进步研究的报告。会议范围包括机器学习、自然语言处理、计算机视觉、数据挖掘、多智能体系统、知识表示、human-in-the-loop AI、搜索、规划、推理、机器人和感知以及伦理。除了专注于这些领域中的任何一个领域的基础工作外，我们明确鼓励跨越人工智能技术领域的工作(例如，机器学习和计算机视觉;计算机视觉与自然语言处理;或机器学习和规划)，人工智能和相关研究领域之间的桥梁(例如神经科学;认知科学)，或者在医疗、可持续发展、交通和商业等重要应用领域开发人工智能技术。

AAAI-24关键字集可在AAAI-24关键字页面上获得。作者的“选择最佳关键词指南”描述了为论文选择关键词时需要考虑的重要因素。

AAAI-24上的大多数论文将成为 Main Track 的一部分。所有主要的跟踪文件将按照相同的标准和通过相同的程序进行审查。本次大会有两个特别的主题，分别是“人工智能的社会影响”和“安全、稳健和负责任的人工智能”。特殊方向的论文将根据与主要方向的论文不同的评估标准进行审查。所有论文将遵循相同的审稿时间表。

## Review Criteria

AAAI是一个高度选择性的会议。因此，我们强烈鼓励未来的作者只向AAAI提交他们最好的作品。

所有提交的论文将由专家评审人员严格评估，以评估论文的贡献是否足够实质性，以保证在AAAI上发表。

这些贡献可能是理论的、方法的、算法的、经验的、综合的(将不同的人工智能子领域的思想和方法联系起来)，或者是批判性的(例如，原则性的分析和论点，将注意力集中在根本错误的目标、假设或方法的选择上)。对提交给“人工智能促进社会影响”专题的材料进行评估时，应强调研究在解决紧迫的社会挑战(例如卫生、食品、环境、教育、治理等)方面所显示的或潜在的影响。对提交给“安全、稳健和负责任的人工智能”轨道的评估应展示出在理论和实践层面上安全、稳健和负责任的人工智能系统方面取得的集中、切实的进展。

所有提交的材料都将根据贡献的意义和新颖性（研究问题或解决的问题、方法、实验、分析）、主张的理论和/或经验合理性、它们与AAAI社区的相关性以及阐述的清晰度进行评估和评分。

其他考虑因素包括遵守负责任的研究实践(例如，关于人体受试者研究，使用敏感数据，以及数据和算法偏差)，以及确保研究结果可重复性的步骤(例如，通过提供详细的证明，记录实验，共享数据和共享代码)。

探索新领域或指出新的研究方向，或引入新问题，解决研究问题，或引入人工智能单个子领域以外感兴趣的方法的坚实的技术论文比推进技术水平的论文更受欢迎，但只能是渐进的，或只在人工智能的一个狭窄的子领域内。
