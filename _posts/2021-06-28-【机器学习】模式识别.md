# 贝叶斯决策理论

统计模式识别： 用概率统计的观点和方法解决模式识别的问题。

贝叶斯决策(统计决策理论)：

- 是统计模式识别的基本方法和基础
- 是 "最有分类器"： 使平均错误率最小


条件：

- 类别数一定， $w_i, i=1, ..., c$ (决策论中把类别称作状态)
- 已知类先验概率和类条件概率密度 $P(w_i), P(x \mid w_i), i=1, ..., c$


## 最小错误率贝叶斯决策

$$\min P(e) = \int P(e \mid x)p(x)dx$$

上式等价于 $\min P(e \mid x) \quad \text{for all } x$
